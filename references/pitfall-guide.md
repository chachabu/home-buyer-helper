# 买房避坑指南

## 一、购房前准备

### 资格与预算
- **购房资格**：确认当地限购政策，是否有购房资格
- **征信报告**：提前拉取个人征信报告，确保无逾期、无不良记录
- **首付来源**：首付款必须是自有资金，避免首付贷、信用贷等违规融资
- **月供测算**：月供不超过家庭月收入的40-50%（含公积金覆盖部分）
- **应急资金**：预留6-12个月月供 + 3-5万元装修/维修备用金

### 选筹逻辑
- **需求排序**：学区 > 通勤 > 房龄 > 面积 > 升值空间（优先级因人而异）
- **板块判断**：优先成熟板块，谨慎买入偏远新区（配套落地周期长）
- **流动性考量**：小户型、学区房流动性最好，大户型、商办类流动性差

## 二、现场看房检查清单

### 房屋基本信息核实
- [ ] **朝向采光**：不同时段看房（上午/正午/傍晚），确认采光时长
- [ ] **户型方正**：观察各房间比例是否合理，有无异形/暗房
- [ ] **通风情况**：开窗测试空气对流，高层注意穿堂风
- [ ] **隔音测试**：关窗听外界噪音，敲墙体/地板检查隔音质量
- [ ] **漏水检查**：屋顶、阳台、卫生间墙面/地面有无水渍、霉斑、起皮
- [ ] **排水测试**：卫生间、厨房、阳台地漏倒水测试排水速度
- [ ] **水电测试**：所有开关、插座、水龙头逐个测试，查看配电箱
- [ ] **管道检查**：管道有无堵塞、漏水、异味，热水器压力是否正常
- [ ] **门窗密封**：窗户能否正常开合、锁闭，密封条有无老化，隔音效果
- [ ] **电梯体验**：高峰时段等电梯时间，电梯运行是否平稳

### 小区与周边环境
- [ ] **物业管理**：门禁是否严格、公共区域清洁度、保安巡逻情况
- [ ] **停车情况**：车位配比、地面/地下、租赁费用
- [ ] **噪音来源**：是否临近主干道、高架、铁路、工地、商铺
- [ ] **安全因素**：小区是否有监控、消防通道是否通畅
- [ ] **学区确认**：确认对应学区/学位是否被占用，落户年限要求
- [ ] **周边配套**：超市、菜市场、医院、公园、餐饮、快递柜等

### 建筑与产权信息
- [ ] **房龄核实**：对比房产证/不动产证上的竣工日期与房屋实际状况
- [ ] **建筑结构**：砖混/框架/剪力墙，判断抗震性与改造空间
- [ ] **物业费/采暖费**：确认金额、支付周期、欠费情况
- [ ] **共有情况**：确认是否有多个产权人，共有人是否同意出售

## 三、产权核验要点

### 必须核验的证件
- [ ] **不动产登记信息查询结果**：确认产权人、坐落、面积、抵押状态
- [ ] **房屋买卖合同**：确认所有产权人均在合同上签字
- [ ] **契税完税证明**：确认税费已缴清
- [ ] **身份证明**：核实卖方身份证与产权证是否一致

### 需确认的状态
- [ ] **是否被抵押**：有无银行贷款未还清（可正常过户但需解押流程）
- [ ] **是否被查封**：涉诉/被执行房源禁止购买
- [ ] **是否设立居住权**：居住权登记后即使过户原居住人仍可居住
- [ ] **是否满五唯一**：满五唯一免增值税+个税，满二免增值税，不满二全额增值税
- [ ] **学位是否被占用**：部分城市学位锁定6年，需提前确认

### 二手房额外核验
- [ ] 前业主是否有户口未迁出（影响学位使用）
- [ ] 原始购房合同/发票（用于核验税费计算基数）
- [ ] 物业交割清单（确认无欠费）

## 四、签约与过户注意事项

### 定金 / 诚意金
- 定金金额不超过总价的20%
- 定金收据写明收款方、金额、用途
- 明确违约责任和定金罚则

### 买卖合同关键条款
- [ ] **成交价格**：明确总价，大小写一致
- [ ] **付款方式**：首付金额及支付时间节点、贷款申请时间
- [ ] **过户时间**：明确办理过户的具体时间节点
- [ ] **税费承担**：明确各税费由谁承担（通常契税买方，增值税/个税卖方）
- [ ] **交房时间**：明确交房日期及交房标准
- [ ] **户口迁出**：卖方户口迁出时间节点，违约金
- [ ] **学位使用**：确认学位未占用或占用情况写入合同
- [ ] **违约责任**：明确逾期付款、逾期过户、逾期交房的违约金
- [ ] **补充条款**：所有口头承诺一律写入补充协议

### 资金安全
- **首付资金监管**：通过银行资金监管账户支付，避免直接打给卖方
- **贷款预审**：签合同前完成银行贷款预审批，确认可贷额度和利率
- **过户当日支付**：剩余首付/全款在过户当日支付至监管账户

### 过户流程
1. 买卖合同网签备案
2. 缴税（契税、增值税、个税等）
3. 不动产登记过户
4. 过户完成后领取新不动产证
5. 领取房产证后办理抵押登记（贷款购房）

### 交房验收
- [ ] 确认卖方物业、水电、燃气、暖气等费用已结清
- [ ] 所有钥匙、门禁卡、智能密码交付
- [ ] 附属设施验收（家具、家电如有赠送需清点确认）
- [ ] 签署《物业交割确认书》
- [ ] 办理水、电、燃气、物业更名手续

## 五、税费速查

### 买方税费

| 税种 | 税率 | 说明 |
|------|------|------|
| 契税 | 1%-3% | 首套≤90㎡:1%; >90㎡:1.5%; 二套≥90㎡:2% |
| 登记费 | 80元 | 住宅 |
| 抵押登记费 | 80元 | 贷款购房 |
| 贷款服务费 | 0-2000元 | 视银行/中介而定 |

### 卖方税费

| 税种 | 税率 | 说明 |
|------|------|------|
| 增值税 | 5.3% | 满二唯一免征，不满全额征收 |
| 个人所得税 | 1%或20% | 满五唯一免征；非唯一可按1%或差额20% |
| 中介费 | 1-3% | 通常由买卖双方协商分担 |

> ⚠️ 各地政策可能有差异，税费以当地税务部门核定为准。

## 六、常见陷阱识别

### 产权类陷阱
- **一房多卖**：核实房源是否已被网签或已售出
- **抵押房风险**：未解押房产无法直接过户，流程长、风险高
- **居住权陷阱**：居住权登记后所有权转移不影响居住权人使用
- **学区房变数**：学区划分每年调整，购买前查当年划分、问未来政策

### 价格类陷阱
- **虚假底价**：中介谎称底价诱导谈判
- **税费转嫁**：口头承诺卖方承担税费，合同未写明变成买方承担
- **装修折旧坑**：高价装修房含大量虚高溢价

### 合同类陷阱
- **阴阳合同**：网签低价逃税，实际成交高价，引发纠纷
- **补充条款缺失**：口头承诺不兑现，维权困难
- **交房条件模糊**：交房时家具家电已撤、装修已破坏

### 贷款类陷阱
- **贷前承诺不兑现**：中介承诺利率/额度，银行审批实际不同
- **解押首付挪用**：卖方借首付还贷后不配合过户
- **经营贷买房**：违规使用经营贷、消费贷作首付，银行追贷

---

## 七、购房后注意事项

- 取得不动产证后定期查看产权状态
- 及时办理房屋保险
- 自住业主建议建立维修基金
- 学区房及时办理子女入学手续
- 保留全套交易凭证（合同、发票、完税证明等）

---

*本指南仅供参考，各地政策差异较大，具体操作请咨询当地不动产登记中心和税务部门。*
